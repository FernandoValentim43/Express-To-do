| Other technologies        | Description | 
| :---                      |    :----:   |  
| Mongoose                  | MongoDB and the Node object modeling  for node   | 
| Nodemon                   | Monitor script    | 


### Setup
- `npm i `
- create .env file inside `/api` with **PORT** and **MONGO_URL**
- `npm start` to start the server
